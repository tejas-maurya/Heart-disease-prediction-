❤️ Heart Disease Prediction Web App
📌 Project Description

The Heart Disease Prediction Web App is a Machine Learning–based application that predicts whether a person is at risk of heart disease based on medical attributes.

This project demonstrates a complete end-to-end ML workflow, including:

Data preprocessing

Exploratory Data Analysis (EDA)

Model training and evaluation

Model serialization using Pickle

Deployment using Streamlit

The application provides an interactive web interface where users can input medical parameters and instantly receive prediction results.

📊 Input Features Used

The model uses the following 13 features:

Age

Sex (1 = Male, 0 = Female)

Chest Pain Type (0–3)

Resting Blood Pressure

Cholesterol

Fasting Blood Sugar (1 = True, 0 = False)

Rest ECG Results (0–2)

Maximum Heart Rate Achieved

Exercise Induced Angina (1 = Yes, 0 = No)

Oldpeak (ST Depression)

Slope of Peak Exercise

Number of Major Vessels (0–3)

Thal
